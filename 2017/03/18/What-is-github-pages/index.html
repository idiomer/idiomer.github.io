What is github pages?

github pages相当于一个虚拟web服务器主机，每个github帐号只能有一个github page。如果你的github帐号是abc，那么你的page就会是abc.github.io。那么怎么开始呢？

  1. register a github：先注册一个github帐号，如帐号名为abc

  1. create a repository：创建一个新的项目仓库，命名为\<username>.github.io，即abc.github.io。

  2. clone the repository：克隆一份到本地

    git clone https://github.com/username/username.github.io

  3. hello world：创建你的主页html文件，如

    cd username.github.io

    echo “Hello world! Welcome to my page!” > index.html

  4. push it：推送本地仓库到github同步

  5. done：用浏览器打开username.github.io。Welcome to my page!
