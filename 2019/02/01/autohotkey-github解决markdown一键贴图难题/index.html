autohotkey+github解决markdown一键贴图难题

配置步骤

https://github.com/wanglong001/ClipMd

参考上述git项目

  1. 先下载安装autohotkey
  2. 建立自己的hexo项目
  3. 配置clipMd，这里做了如下修改
    1. PicPath不是放在原项目作者认为的theme/目录里面，因为每个theme的images子目录不一定一样，所以切换主题的时候会出问题，需要重新配置，比较麻烦。所以直接放在了resource/目录下
    2. 由于第1点的修改，所以httpPath也要相应地修改成https://raw.githubusercontent.com/${your_git_name}/${your_git_name}.github.io/master/images/；其实不修改也没问题，但是如果绑了自己的域名会有问题，所以建议改成这样。
