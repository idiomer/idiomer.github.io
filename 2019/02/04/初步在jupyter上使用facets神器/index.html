初步在jupyter上使用facets神器

背景

​ 昨天在逛知乎的时候，偶然发现了facets这个可视化的神器。然后去web版体验了一下功能，非常棒：overview能快速对自己的数据集有一个快速且全面的了解，进而能快速地发现其中的异常数据；dive的功能更是炫丽，通过图表+缩放的逆天前端实现，直观快速地了解多维度组合下的特征区分度。OK! Talk is cheat, show me the picture

​ 虽然web版可以直接使用，但是考虑到数据安全性的关系，特别是企业业务的关键数据，在自己的jupyter上搭建facets的可控性是最好的，并且可以灵活处理数据。

安装步骤

前提：安装好了jupyter

  1. 安装protobuf：pip3 install protobuf
  2. 安装facets：
    1. git clone https://github.com/PAIR-code/facets
    2. cd facets/
    3. jupyter nbextension install facets-dist/ --user
  3. 测试是否正常运行
    1. 开启jupyter后，运行facets/facets_overview/Overview_demo.ipynb这个notebook，看是否成功
    2. 看到能正常出现overview的界面即可（注意：当前只支持chrome浏览器）
    3. 之后按这个demo的notebook，换成自己的数据集即可
